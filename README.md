# Subscription UI

This project is a subscription management UI built with React and TypeScript.

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Vite

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/bhupenderkumar/subscription-ui.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

## Project Structure

- `src/`: Contains the source code of the application.
  - `components/`: Contains the React components.
  - `App.tsx`: The main application component.
  - `index.css`: Global styles.
